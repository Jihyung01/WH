## 1. 현재 구현 상태 총정리

## 2. 단기 개선 (1~2개월 — 현재 스택으로 가능)

### 2-1. 오프라인 캐싱 & 성능
- **이벤트 데이터 로컬 캐싱**: AsyncStorage 또는 MMKV로 최근 조회 이벤트 캐싱. 지하철/건물 내부에서도 마지막 데이터 표시
- **지도 마커 클러스터링**: 이벤트가 많아지면 마커 겹침 → `react-native-map-clustering`으로 줌 레벨별 자동 그룹화
- **이미지 프리페칭**: expo-image의 `prefetch`로 스크롤 전 이미지 미리 로드
- **근거**: 지하철 이동 중 앱을 켜는 사용자가 많음. 오프라인에서 "아무것도 안 보임" → 이탈률 증가

### 2-2. 소셜 기능 v1
- **친구 시스템**: profiles 테이블에 friend 관계 테이블 추가, 친구의 탐험 활동 피드
- **탐험 공유**: 완료한 이벤트를 카카오톡/인스타 스토리로 공유 (딥링크 포함)
- **실시간 친구 위치**: Supabase Realtime으로 파티원 위치 표시 (옵트인)
- **근거**: 위치 기반 앱의 핵심 바이럴 채널. 포켓몬GO도 친구 시스템 추가 후 DAU 30% 증가

### 2-3. 알림 고도화
- **지오펜싱 알림**: 이벤트 500m 접근 시 "근처에 탐험 포인트가 있어요!" 자동 푸시
- **이벤트 마감 알림**: 종료 24시간 전 "내일 마감되는 이벤트가 있어요"
- **일일 탐험 리마인더**: 오후 5시 "오늘의 탐험을 시작해보세요" (개인화)
- **근거**: 푸시 알림은 DAU 유지의 핵심. 지오펜싱은 "우연한 발견" 경험을 만듦

### 2-4. 온보딩 개선
- **인터랙티브 튜토리얼**: 첫 로그인 시 가이드 오버레이 (체크인 방법, 미션 설명)
- **첫 이벤트 자동 추천**: GPS 기반으로 가장 가까운 이벤트 하이라이트
- **캐릭터 커스터마이징 확장**: 색상 변경, 액세서리 (현재 4종 → 12종+)
- **근거**: 첫 30분 내 코어루프 완료 여부가 D1 리텐션 결정 (업계 평균 25%, 목표 40%)

### 2-5. 미션 다양화
- **AR 사진 미션**: expo-camera로 특정 랜드마크 인식 또는 AR 오버레이
- **타임어택 미션**: 제한 시간 내 여러 체크포인트 방문
- **연속 미션 체인**: 3개 장소를 순서대로 방문하는 스탬프 랠리
- **근거**: 미션 다양성 부족 시 반복 피로감 → 2주 차 이탈 급증

## 3. 중기 확장 (3~6개월 — 새 기능/서비스 추가)

### 3-1. 시즌/이벤트 시스템
- **시즌 패스**: 30일 단위 시즌, 레벨업 보상 트랙 (무료/프리미엄)
- **한정 이벤트**: 벚꽃 시즌, 크리스마스, 대학 축제 기간 특별 이벤트
- **시즌 리더보드**: 시즌별 초기화, 상위 랭커 보상 (한정 배지/스킨)
- **근거**: 시즌제는 지속적 복귀 동기 부여. Fortnite/원신 모델 검증됨. ₩4,900 구독 외 시즌패스 ₩9,900 추가 수익

### 3-2. 크루/길드 시스템
- **크루 생성**: 최대 20명, 크루 전용 이벤트/미션
- **크루 대항전**: 주간 크루별 포인트 경쟁
- **크루 영역**: 특정 동네를 크루 홈으로 지정, 영역 포인트 누적
- **근거**: 소셜 바인딩 = 이탈 방어. 크루 소속 유저의 리텐션은 비소속 대비 2~3배

### 3-3. UGC (User Generated Content)
- **유저 이벤트 생성**: 일반 유저가 자기 동네에 이벤트 등록 (관리자 승인제)
- **이벤트 평가/리뷰**: 별점 + 한줄평, 인기 이벤트 추천 알고리즘
- **포토 갤러리**: 미션 사진 모아보기, 좋아요/댓글
- **근거**: 콘텐츠 공급 병목 해소. 운영팀이 30개 도시 이벤트를 직접 만들 수 없음

### 3-4. 결제 시스템
- **인앱 결제**: expo-in-app-purchases 또는 RevenueCat 연동
- **프리미엄 구독 (₩4,900/월)**: XP 2배, 전용 이벤트, 광고 제거, 프리미엄 스킨
- **코인 구매**: 특별 아이템, 부스터 구매용 인앱 화폐
- **근거**: 수익화 없이 서비스 지속 불가. 구독 모델은 LTV 예측 가능

### 3-5. B2B 제휴 플랫폼
- **사업자 대시보드 (웹)**: Next.js로 제휴 업체용 관리 페이지
- **제휴 이벤트 등록**: 카페/음식점/관광지가 직접 이벤트 생성 + 쿠폰 연동
- **분석 리포트**: 방문 수, 체크인 수, 전환율 데이터 제공
- **근거**: 핵심 B2B 수익원. 로컬 소상공인 마케팅 니즈 + 관광청/지자체 협업 가능

### 3-6. AI 고도화
- **개인화 추천**: 탐험 이력 기반 이벤트 추천 (collaborative filtering + Claude)
- **AI 가이드 캐릭터**: 채팅형 탐험 가이드 (캐릭터 페르소나별 다른 말투)
- **동적 난이도**: 사용자 레벨/완료율에 따라 퀴즈 난이도 자동 조절
- **근거**: AI 차별화 = 경쟁 우위. 일반 스탬프 앱과의 핵심 차이점

### 4-3. 오픈 플랫폼
- **이벤트 API 공개**: 외부 개발자가 WhereHere 이벤트를 자기 앱에 연동

- **크리에이터 이코노미**: 이벤트 제작자가 수익 쉐어

### 4-4. 스마트시티 연동
- **공공 데이터 통합**: 서울 열린데이터광장, 관광공사 API 자동 연동
- **대중교통 연계**: 이벤트까지 지하철/버스 경로 안내
- **지자체 협업**: 관광 활성화 프로그램으로 B2G 수익

## 5. 기술적 개선 (성능, 아키텍처, 인프라)

### 5-1. 즉시 개선 가능
- **MMKV 도입**: AsyncStorage → react-native-mmkv (10배 빠른 읽기/쓰기)
- **React Query (TanStack Query)**: API 캐싱, 자동 재시도, 옵티미스틱 업데이트
- **이미지 최적화**: Supabase Storage 이미지 변환 (썸네일 자동 생성, WebP)
- **번들 크기 최적화**: Metro bundler 트리 쉐이킹, lazy import

### 5-2. 중기 아키텍처
- **Supabase Realtime**: 실시간 이벤트 참여자 수, 친구 위치, 라이브 리더보드
- **Background Location**: expo-task-manager로 백그라운드 위치 추적 (지오펜싱)
- **에러 추적**: Sentry 연동 (크래시 리포트, 성능 모니터링)
- **분석**: Mixpanel 또는 Amplitude (유저 행동 분석, 퍼널 추적)
- **CI/CD**: GitHub Actions → EAS Build 자동화 (PR 머지 시 preview 빌드)

### 5-3. 장기 인프라
- **Edge Computing**: Supabase Edge Functions을 Deno Deploy로 글로벌 분산
- **CDN**: 이미지/에셋 CloudFlare CDN 캐싱
- **Database 읽기 전용 복제본**: 리더보드/통계 쿼리 분산
- **모니터링 대시보드**: Grafana + Supabase 메트릭스로 실시간 서비스 상태

## 8. 경쟁 차별화 아이디어

### 8-1. 현재 경쟁 환경
| 경쟁자 | 특징 | WhereHere 차별점 |
|--------|------|------------------|
| 포켓몬GO | AR 수집, 글로벌 | AI 서사 + 실제 장소 문화 콘텐츠 |
| 카카오맵 스탬프 | 단순 체크인 | 게이미피케이션 + 캐릭터 성장 |
| 네이버 MY플레이스 | 리뷰/방문 기록 | 미션 기반 인터랙티브 경험 |
| 스탬프투어 앱들 | 관광지 스탬프 | AI 동적 콘텐츠 + 소셜 경쟁 |

### 8-2. 핵심 차별화 전략
1. **AI 서사 엔진**: 모든 장소에 고유한 스토리. 같은 장소도 날씨/시간/유저 레벨에 따라 다른 서사 → "매번 다른 경험"
2. **캐릭터 성장 RPG**: 단순 스탬프가 아닌 캐릭터 육성. 탐험할수록 강해지는 느낌
3. **동적 퀴즈**: AI가 장소 기반 퀴즈를 실시간 생성 → 교육적 가치 + 재미
4. **크루 영역 시스템**: "우리 동네는 우리 크루 것" — 영역 지배 경쟁 (Ingress 영감)
5. **로컬 소상공인 연계**: 체크인 → 쿠폰 자동 발급. 다른 위치 앱에 없는 실질 혜택

### 8-3. 방어 가능한 해자 (Moat)
- **데이터 축적**: 유저 행동 데이터 + 장소 데이터 → 개인화 추천 강화
- **네트워크 효과**: 크루/친구 → 유저가 많을수록 가치 상승
- **UGC 콘텐츠**: 유저 생성 이벤트가 쌓일수록 콘텐츠 풍부
- **B2B 관계**: 제휴 업체가 많을수록 이벤트 다양 → 유저 유입 → 더 많은 제휴
